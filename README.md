## Get started
0. you just star the repositry and fork it.
1. clone the forked repositry using 
   ```bash
   git clone "forked_repositry_url"
   ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
    npm start
   ```
